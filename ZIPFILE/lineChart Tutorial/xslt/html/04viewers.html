How to view SVG files in your browser

  • If you are using Internet Explorer then you will need to download and install the Adobe SVG Viewer for Windows. There is a Macintosh version of the Adobe SVG Viewer as well.

    Once installed, SVG content will be rendered by the plugin.
  • If you are using Mozilla 1.7 there will eventually be SVG support built in, but at present there is no support.

    In this case it may be best to transform the SVG file to an image file (JPEG, TIFF or PNG) using the Batik Rasterizer.

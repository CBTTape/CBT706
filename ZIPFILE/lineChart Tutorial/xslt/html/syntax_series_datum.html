<datum>

Syntax

<datum x="x-value" y="y-value x2="x2-value" y2="y2-value">
  annotation text
</datum>

Synopsis

A series consists of one or more data points. Each data point is defined by a <datum> tag.

Examples:

  • A missing value that is not to be plotted (there will be a gap in the line here):

    <datum/>
  • A specific y-value for the next integral x-value:

    <datum y="3.43"/>
  • A specific x- and y-value pair:

    <datum x="10" y="3.43"/>
  • A point in a line that has an annotation (text that is displayed near that point):

    <datum x="10" y="3.43">Peak loading</datum>
  • A point whose y-value appears on the secondary y-axis:

    <datum x="10" y2="447"/>
  • A point plotted against the secondary x- and y-axis:

    <datum x2="10" y2="447"/>

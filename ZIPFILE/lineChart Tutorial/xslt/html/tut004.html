Add a legend to the east pane

By default a legend in the east pane will have one column. This can be changed by specifying the <layout> tag.

Chart xml sourceResulting image
<chart>
    <title>Weather Balloon Tracking</title>

    <series>
        <datum y="1"/>
        <datum y="3"/>
        <datum y="9"/>
        <datum y="8"/>
        <datum y="7"/>
    </series>

    <series>
        <datum y="11"/>
        <datum y="117"/>
        <datum y="87"/>
        <datum y="63"/>
        <datum y="2"/>
    </series>
    
    <layout>
        <east>
            <legend/>
        </east>
    </layout>
    
</chart>

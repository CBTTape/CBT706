Create a new XML file containing a <chart> tag

All other tags are contained by the <chart> tag. Optionally, you can specify a debug attribute. When set to "true", extra comments will be generated in the resulting svg file that may be helpful.

<?xml version="1.0" encoding="utf-8"?>
<chart debug="true">

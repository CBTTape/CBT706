How to Create Line Charts from XML Data

This shows you how to create a Scalable Vector Graphics (SVG) line chart from an XML input file so that a user can view it in a web browser. You can optionally convert the SVG file to an image file.

The Problem

You have data that you need to publish as a line chart and be viewable by users using their web browser. A typical example is having to publish daily CPU usage figures in graphical form on your intranet.

A Solution

This solution involves the following steps:
  1. Extract the data that you want to chart and use it create a text file with a simple XML format. The XML file describes your chart data.
  2. Transform that simple XML format into the more complex SVG format using an XSLT processor (for example Xalan) and an XSLT stylesheet (supplied by this solution). The resulting SVG file describes the chart drawing.
  3. Publish the SVG data directly to your web server (or convert that SVG into an image file first and publish the image).
  4. Browse the SVG (or image) using a web browser.

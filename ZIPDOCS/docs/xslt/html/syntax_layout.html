<layout>

Syntax

<layout>
    <width>800</width>
    <height>600</height>
    <margin>10</margin>
    <title>
        <height>40</height>
    </title>
    <north>
        <title>
            <height>20</height>
        </title>
        <label>
            <height>10</height>
        </label>
    </north>
    <south>
        <label>
            <fontsize>10</fontsize>
            <rotation>-90</rotation>
            <height>10</height>
        </label>
        <title>
            <height>20</height>
        </title>
        <legend>
            <height>60</height>
            <columns>4</columns>
        </legend>
    </south>
    <east>
        <label>
            <width>50</width>
        </label>
        <title>
            <width>20</width>
        </title>
        <legend>
            <columns>1</columns>
        </legend>
    </east>
    <west>
        <title>
            <height>20</height>
        </title>
        <label>
            <width>50</width>
            <height>10</height>
        </label>
    </west>
</layout>

Synopsis

The <layout> tag describes the main chart areas. See Overview of chart layout for more information.

The chart is divided into the following areas:

North

West

Under the <layout> tag you can insert:

  • a <width> tag specifying the width of the chart. By default units are pixels, but you can specify real world units such as 200mm if you like.
  • a <height> tag specifying the height of the chart.
  • a <margin> tag specifying the size of the margin surrounding the entire chart (like the white margin around a photograph). This is deducted from the chart width and height, not added to it.
  • a <title> tag specifying the title of the chart.
  • a <north> tag describing the North area.
  • an <east> tag describing the East area.
  • a <south> tag describing the South area.
  • a <west> tag describing the West areas.
Under each of the <north>, <east>, <south> and <west> tags you can insert:
  • a <title> tag specifying space to reserve (width or height) for the axis title.
  • a <label> tag specifying space to reserve (width or height) for the axis labels. In addition, you can specify the rotation to be applied to each label by specifying a <rotation> tag under the <label> tag, and the height of the label text (in pixels) by specifying the <fontsize> tag under the <label> tag.
  • a <legend> tag to indicate whether you want a legend to appear in this area and optionally the space to reserve for it (width or height). 
  • any number of <reference> tags specifying that a reference line is to be drawn at a particular place on this axis. This is handy when you need to show minimum or maximum Y-values or perhaps a milestone X-value.
  • any number of <label> tags to specify the text (rather than a numeric value) to appear at major gridline intervals on this axis. If you omit the <label> tags, then appropriate numeric labels will be generated. 

The current implementation only allows a legend to appear in the east or south areas. Under the <legend> tag you can insert:

  • A <columns> tag to specify how many columns the legend is to occupy. You would typically use a single column for a legend in the east area, but two or more columns in the south area.

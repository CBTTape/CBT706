How to transform XML data into SVG

  • If you want to transform the XML file into SVG on a PC then make sure you have downloaded and installed Sun's Java Runtime Environment
  • If you want to transform the XML file into SVG on an IBM z/Series mainframe, then make sure you have downloaded and installed IBM's Java Development Kit 1.4 for z/Series
  • Download the Apache Project's XALAN XSLT processor
  • Put your chart data into a simple XML file format. Because the XML input file is plain text, it can be created by any program.
  • Run the following command to transform the XML input file (your.xml) into SVG format (your.svg) using the stylesheet called lineChart.xsl: 
    java org.apache.xalan.xslt.Process -in your.xml -xsl lineChart.xsl -out your.svg -param protocol http:

    Note: The "-param protocol http:" is a hack to make the Adobe SVG Viewer plugin handle relative URLs properly. You only need this if you are using the Adobe SVG Viewer 3.01 plugin and are going to serve the SVG out from a web server.

Turn on the debug option

You can add the attribute debug="true" to the <chart> tag to see how big each area is. This will draw a faded rendition of the chart and then overlay it with rectangles showing the current size of each area using a "width x height" notation. It will also show the height, in pixels, of the font used in each area. From this, we can see that increasing the X-axis label area height from 20 (the default) to 40 pixels should provide sufficient space for the rotated labels. We could instead reduce the height of the label font by adding a <fontsize> tag under the <labels> tag.

    <series>

        <title>Scorpio</title>
        <datum y2="11"/>
        <datum y2="117"/>
        <datum y2="87"/>
        <datum y2="63"/>
        <datum y2="2"/>
    </series>

    <layout>
        <south>
            <legend/>
            <labels>
                <rotate>-90</rotate>
            </labels>
        </south>
    </layout>

    <axis>
        <y>
            <title>Aquarius Height Above Sea Level (km)</title>
            <range>
                <min>0</min>
                <max>15</max>
            </range>
            <interval>
                <major>5</major>
            </interval>
        </y>
        <y2>
            <title>Scorpio Height Above Sea Level (km)</title>
            <range>
                <min>0</min>
                <max>150</max>
            </range>
        </y2>
        <x>
            <title>Time Since Launch</title>
            <label>00:00</label>
            <label>01:00</label>
            <label>02:00</label>
            <label>03:00</label>
            <label>04:00</label>
        </x>
    </axis>

</chart>

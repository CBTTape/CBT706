Add a series of data points to be plotted

Each series consists of one or more <datum> tags. Each <datum> tag can have an x attribute (or an x2 attribute for the secondary X-axis) and a y attribute (or a y2 attribute for the secondary Y-axis). You can plot missing values by simply omitting the Y- value. This will cause a gap to appear in the line. You can also omit X- values in which case a suitable integral value will be assigned.

Notice how the Y-axis range is determined automatically to encompass the minimum and maximum y-values to be plotted. This can be changed later.

Chart xml sourceResulting image
<chart>
    <title>Weather Balloon Tracking</title>
    
    <series>
        <datum y="1"/>
        <datum y="3"/>
        <datum y="9"/>
        <datum y="8"/>
        <datum y="7"/>
    </series>
    
</chart>

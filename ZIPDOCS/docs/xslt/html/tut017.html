Tutorial

Tile the image over the plotarea

By default the image is tiled within the major gridline boundaries. Note: Tiling support appears to be broken in Adobe SVG Viewer 3.01 (and v6 beta), but works well with the Apache Project's Batik SVG implementation in Java. For the remainder of this tutorial we will not tile the background image.

Chart xml sourceResulting image
<chart>
    <title>Weather Balloon Tracking</title>

    <series>
        <title>Aquarius</title>
        <datum y="1"/>
        <datum y="3"/>
        <datum y="9"/>
        <datum y="8"/>
        <datum y="7"/>
    </series>

    <series>
        <title>Scorpio</title>
        <datum y2="11"/>
        <datum y2="117"/>
        <datum y2="87"/>
        <datum y2="63"/>
        <datum y2="2"/>
    </series>

    <style>
        <plotarea>
            <picture opacity="0.5" location="tile">background.jpg</picture>
        </plotarea>
    </style>

    <layout>
        <south>
            <legend/>
            <labels>
                <rotate>-90</rotate>
                <height>40</height>
            </labels>
        </south>
    </layout>

    <axis>
        <y>
            <title>Aquarius Height Above Sea Level (km)</title>
            <range>
                <min>0</min>
                <max>15</max>
            </range>
            <interval>
                <major>5</major>
            </interval>
        </y>
        <y2>
            <title>Scorpio Height Above Sea Level (km)</title>
            <range>
                <min>0</min>
                <max>150</max>
            </range>
        </y2>
        <x>
            <title>Time Since Launch</title>
            <label>00:00</label>
            <label>01:00</label>
            <label>02:00</label>
            <label>03:00</label>
            <label>04:00</label>
        </x>
    </axis>

</chart>

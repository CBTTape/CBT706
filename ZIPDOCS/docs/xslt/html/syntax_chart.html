<chart>

Syntax

<chart>
    <title>Your Chart Title</title>

    <series> <!-- The first data series to be plotted -->
    </series>

    <series> <!-- The second data series to be plotted -->
    </series>

        .
        .
        .

    <series> <!-- The nth data series to be plotted -->
    </series>

    <axis>
    </axis>

    <layout>
    </layout>

    <style>
    </style>
</chart>

Synopsis

The <chart> tag is the outermost tag in the document. Currently, there is only one attribute supported and that is debug. Specify <chart debug="true"> if you want debugging information embedded as comments in your svg output file.

Under the <chart> tag you can insert:

  • a <title> tag describing the chart title
  • a <series> tag for each line to be plotted
  • an <axis> tag describing the scale and range of each axis
  • a <layout> tag describing sizes of various areas of the chart
  • a <style> tag describing the styles (colors, fonts etc) that you want to apply to the chart.

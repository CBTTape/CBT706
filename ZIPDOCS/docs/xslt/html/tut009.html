Adjust the Y-axis major grid line interval

Set the interval between y-axis major grid lines to 5. Note that the grid lines are always drawn based on the Y-axis interval setting rather than the Y2-axis interval setting - otherwise you would end up with separate grid lines for each axis.

Chart xml sourceResulting image
<chart>
    <title>Weather Balloon Tracking</title>

    <series>
        <title>Aquarius</title>
        <datum y="1"/>
        <datum y="3"/>
        <datum y="9"/>
        <datum y="8"/>
        <datum y="7"/>
    </series>

    <series>
        <title>Scorpio</title>
        <datum y2="11"/>
        <datum y2="117"/>
        <datum y2="87"/>
        <datum y2="63"/>
        <datum y2="2"/>
    </series>

    <layout>
        <south>
            <legend/>
        </south>
    </layout>

    <axis>
        <y>
            <range>
                <min>0</min>
                <max>15</max>
            </range>
            <interval>
                <major>5</major>
            </interval>
        </y>
        <y2>
            <range>
                <min>0</min>
                <max>150</max>
            </range>
        </y2>
    </axis>

</chart>

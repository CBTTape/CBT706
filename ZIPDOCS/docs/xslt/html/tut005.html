Move the legend to the south pane

By default the south pane has four columns. You can change this by adding a <columns> element under the <legend> element.

Chart xml sourceResulting image
<chart>
    <title>Weather Balloon Tracking</title>

    <series>
        <datum y="1"/>
        <datum y="3"/>
        <datum y="9"/>
        <datum y="8"/>
        <datum y="7"/>
    </series>

    <series>
        <datum y="11"/>
        <datum y="117"/>
        <datum y="87"/>
        <datum y="63"/>
        <datum y="2"/>
    </series>

    <layout>
        <south>
            <legend/>
        </south>
    </layout>

</chart>

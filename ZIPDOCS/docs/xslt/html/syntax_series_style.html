Using the "style" Attribute

Syntax

style="name:value;name:value;...;"

Synopsis

You can override the default style (for example, line color, font name etc) by adding a style attribute. You can specify one or more of the CSS2 Style Attributes as described in the "Styling" chapter of the SVG Specification.

For example, a data series may have the following style attribute specified on its <series> tag:

style="stroke:orange;stroke-width:2;stroke-dasharray:10,5;filter:none"

This means:

  • Make the line (stroke) color orange. Orange is one of the many pre-defined SVG color names.
  • Make the line width (stroke-width) 2 pixels wide.
  • Make the line style a sequence of 10-pixel dashes with 5-pixel gaps.
  • Remove the filter that would have applied a drop-shadow effect to the text (as defined in lineChart.xsl).

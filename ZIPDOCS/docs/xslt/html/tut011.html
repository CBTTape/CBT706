Add X-axis labels

By default, numeric axis labels are generated for you.  Each label is positioned near a major grid line. For numeric labels, you can specify your own format by specifying a <format> tag under <style><..area..><label><format>xxx</format>. The format you specify in xxx is anything acceptable to the XSLT format-number function. For example, 0###.##, means pad on the left with leading zeros to a width of four digits and display up to two decimal places. You can specify your own labels by adding <label> tags - these a treated as text and are not formatted as numbers. In this tutorial, we will specify our own X-axis labels to display hours and minutes in hh:mm format:

Chart xml sourceResulting image
<chart>
    <title>Weather Balloon Tracking</title>

    <series>
        <title>Aquarius</title>
        <datum y="1"/>
        <datum y="3"/>
        <datum y="9"/>
        <datum y="8"/>
        <datum y="7"/>
    </series>

    <series>
        <title>Scorpio</title>
        <datum y2="11"/>
        <datum y2="117"/>
        <datum y2="87"/>
        <datum y2="63"/>
        <datum y2="2"/>
    </series>

    <layout>
        <south>
            <legend/>
        </south>
    </layout>

    <axis>
        <y>
            <title>Aquarius Height Above Sea Level (km)</title>
            <range>
                <min>0</min>
                <max>15</max>
            </range>
            <interval>
                <major>5</major>
            </interval>
        </y>
        <y2>
            <title>Scorpio Height Above Sea Level (km)</title>
            <range>
                <min>0</min>
                <max>150</max>
            </range>
        </y2>
        <x>
            <title>Time Since Launch</title>
            <label>00:00</label>
            <label>01:00</label>
            <label>02:00</label>
            <label>03:00</label>
            <label>04:00</label>
        </x>
    </axis>

</chart>

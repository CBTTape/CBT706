Create a new XML file containing a <chart> tag

All other tags are contained by the <chart> tag. Optionally, you can specify a debug attribute. When you specify "debug=true", extra comments will be generated in the resulting svg file and rectangles showing the sizes of the chart areas are drawn. We will use this feature later in the tutorial. You would normally run the transformation with debug="false" or omit the debug attribute altogether.

Chart xml source Resulting image 
<?xml version="1.0" encoding="utf-8"?>
<chart debug="false">
